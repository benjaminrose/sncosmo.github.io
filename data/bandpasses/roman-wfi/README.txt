# The Nancy Grace Roman, Wide Filed Instrument Photometric Filters

* 2021-03 values - NASA's projected values as of 2020.11.30, obtained from Jeff Kruk on 2021.03.12 (Roman_effarea_20201130.txt)
* Previous filter definitions can be found in Hounsell et al. 2018 (https://ui.adsabs.harvard.edu/abs/2018ApJ...867...23H)

![Image summarizing filters. See roman.gsfc.nasa.gov/instruments_and_capabilities.html for latest values.](https://roman.gsfc.nasa.gov/images/roman/100Percent/Roman-Instruments-Chart.jpg)

## Colloquial names

* R - f062
* Z - f087
* Y - f106
* J - f129
* H - f158
* F - f184
* K-short - f213
* Wide - f146
